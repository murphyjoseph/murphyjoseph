# Hey, I'm Joe Murphy (aka Murph).

## What I Care About

**Clarity.**

Most of my work comes down to the same instinct: find the thing that's more complicated than it needs to be, and make it simpler. Sometimes that's a tangled frontend architecture slowing a team down. Sometimes it's a product conversation where nobody's said out loud what we're actually building.

## What I Build

I build with **React, TypeScript, and Next.js**, usually in complex domains like healthcare and marketplaces where getting it wrong has real consequences. I work across the stack through Python and Django when I need to, but my home base is where system design meets user experience.

I co-founded **[Mitobyte](https://www.mitobyte.com)**, a developer community built around the idea that mentorship and confidence matter as much as technical skill.
